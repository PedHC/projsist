O serviço de rodízios é bastante popular no país, mas não é fácil encontrar todas as opções disponíveis em uma cidade, principalmente uma capital.


RodEasy veio para resolver este problema, trazendo uma plataforma simples para procura e reserva de rodízios em sua região.

Arquitetura do Software




O sistema E-Bico mediará a negociação entre as duas bases de usuários: Clientes e Autônomos, lidando principalmente com a busca de profissionais e com a logística de pagamento, através de um provedor parceiro.


A interface de acesso dos usuários se dá através do aplicativo para dispositivos móveis e do website, ambos utilizando da plataforma React, com acesso ao sistema interno estabelecido em Node.js através de padrões seguros de comunicação.
Os dados dos clientes e o sistema de busca presente no aplicativo serão armazenados e executados através do banco de dados Neo4j, que utiliza do modelo de grafos relacional, nos permitindo assim utilizar de algorítmos de busca maduros e confiáveis.
